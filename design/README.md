![](./concept.jpg)
>仿造淘宝平台设计的简易电商平台，账号登陆后进入主页，通过主页快捷键可进入包括分类、品牌专区、专题等四个专属模块，大数据化你的购物喜好。看到喜欢的商品可以点进去查看大图、购买和加入购物车。还可以进入消息界面与客服咨询相关内容、查看个人主页。
